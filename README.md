### 1. Clone Repository


### 2. Install TypeScript

Jika TypeScript belum terinstall, jalankan:

```sh
npm install -g typescript
```

### 3. Jalankan Kode

Kompilasi TypeScript ke JavaScript:

```sh
tsc tugas.ts
```

Lalu jalankan hasil kompilasi dengan Node.js:

```sh
node tugas.js
```
